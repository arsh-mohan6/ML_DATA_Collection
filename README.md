#  Machine Learning Data Collection

This project demonstrates simple methods of *data collection* for Machine Learning.

##  Methods Used
### 1️ Manual Data
Manually created dataset using Python.

### 2️ Web Scraping
Data collected from [Quotes to Scrape](https://quotes.toscrape.com) using:
- requests
- BeautifulSoup4

Output: web_scraped_data.csv

### 3️ API Data
Collected pokeman info from [Pokemon API](https://pokeapi.co/api/v2/pokemon).

##  Tools Used
- Python  
- Requests  
- BeautifulSoup4  
- Pandas

### 4 IMDB Open DataSet 
I  am gathering data from the open IMDB dataset using the Hugging Face datasets library.  
The data is converted into a CSV file and the first 5 rows are displayed.

## Steps
1. Load IMDB dataset  
2. Convert train part to CSV  
3. Print first 5 rows  

## Output
- File: imdb_dataset.csv
- Shows sample data from IMDB reviews
